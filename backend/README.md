## Discord Clone Backend

Express and Socket.io server for handling real-time messaging using Prisma as the ORM.

### Prerequisites

**Node version 18.x.x**

### Install packages

```shell
npm i
```

### Setup .env file


```js
DATABASE_URL=
AUTH_SECRET=
```

### Start the app

```shell
npm run start
```

## Available commands

Running commands with npm `npm run [command]`

| command     | description              |
|:------------|:-------------------------|
| `start`     | Starts the app           |
| `build`     | Builds the app           |
